# 📦 PBL MAGAZYN

## 🚀 Getting Started

### Prerequisites

To build and run this project, you will need:

- A C++ compiler (GCC, Clang, MSVC, etc.)
- Git

### Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/lukasz-strama/magazyn.git
   cd magazyn
   ```
