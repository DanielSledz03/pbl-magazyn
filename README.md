# 📦 PBL MAGAZYN

## 🚀 Rozpoczęcie

### Wymagania

Aby zbudować i uruchomić ten projekt, będziesz potrzebować:

- Kompilator C++ (GCC, Clang, MSVC itp.)
- Git

### Instalacja

1. **Sklonuj repozytorium:**

   ```sh
   git clone https://github.com/lukasz-strama/magazyn.git
   cd magazyn
   ```
